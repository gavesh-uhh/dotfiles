# dotfiles

Arch + Hyprland. Catppuccin Mocha.

## Install

```sh
git clone https://github.com/gavesh-uhh/dotfiles.git ~/dotfiles
~/dotfiles/install.sh
```

Symlinks `hypr` `waybar` `wofi` `kitty` into `~/.config` and `.zshrc` into `~`.
Existing files are backed up to `*.bak`.

## Deps

```
hyprland hyprpaper hyprlock hypridle waybar wofi kitty dunst
zsh starship zoxide eza bat fzf
```
